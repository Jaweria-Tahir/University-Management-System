# University-Management-System

  A University Management System developed using Python, PyQt, and SQL, designed to automate university activities such as assignment management, attendance tracking, and GPA calculation.

 Features
-  Assignment upload, download, and submission
-  Attendance management system
-  GPA calculation (course & semester)
-  User profile management with profile picture upload
-  Secure role-based login (Admin, Teacher, Student)
-  Client–Server architecture

 Tools & Technologies
- Programming Language: Python
- GUI Framework: PyQt
- Database: SQL
- Architecture: Client–Server
- Database Concepts:
  - Stored Procedures
  - Indexing
  - Transactions
  - Views
